  <files>
    src/inventor_exporter/model/body.py
  </files>
  <action>
Create Body dataclass that composes foundation types (MODEL-02).

**Body dataclass:**
- `@dataclass(frozen=True)` for immutability
- Required fields:
  - `name: str` - unique identifier (will be sanitized)
  - `transform: Transform` - pose in world frame
- Optional fields:
  - `material_name: Optional[str] = None` - reference to material in AssemblyModel
  - `inertia: Optional[Inertia] = None` - mass properties (from Inventor extraction)
  - `geometry_file: Optional[Path] = None` - path to STEP file

**Name sanitization in __post_init__:**
- Replace ":" with "_" (Inventor uses colons in occurrence names)
- Replace " " with "_" (spaces cause issues in some formats)
- Use `object.__setattr__(self, 'name', sanitized)` since frozen=True

**Validation in __post_init__:**
- Name cannot be empty (after sanitization)
- Raise ValueError with clear message if empty

**Imports:**
- `from pathlib import Path`
- `from typing import Optional`
- Import Transform and Inertia from same package (relative imports)

**Note:** Body does NOT validate that material_name exists - that's AssemblyModel's job since it has the materials collection.
  </action>
  <verify>
```bash
python -c "
from inventor_exporter.model.body import Body
from inventor_exporter.model import Transform, Inertia
from pathlib import Path
import numpy as np

# Create body with just name and transform
t = Transform(position=np.array([1, 2, 3]))
b = Body(name='part1', transform=t)
assert b.name == 'part1'
assert b.transform.position[0] == 1

# Name sanitization
b2 = Body(name='Part:1 Name', transform=Transform())
assert b2.name == 'Part_1_Name', f'Expected Part_1_Name, got {b2.name}'

# With optional fields
b3 = Body(
    name='full_body',
    transform=Transform(),
    material_name='steel',
    inertia=Inertia(mass=1.5),
    geometry_file=Path('parts/body.stp')
)
assert b3.material_name == 'steel'
assert b3.inertia.mass == 1.5
assert b3.geometry_file == Path('parts/body.stp')

# Empty name rejected
try:
    Body(name='', transform=Transform())
    assert False, 'should raise for empty name'
except ValueError as e:
    assert 'empty' in str(e).lower() or 'name' in str(e).lower()

print('Body dataclass: OK')
"
```

<task type="auto">
  <name>Task 2: Create AssemblyModel dataclass with validation</name>
  <files>
    src/inventor_exporter/model/assembly.py
  </files>
  <action>
Create AssemblyModel dataclass - the complete intermediate representation (MODEL-01, MODEL-06).

**AssemblyModel dataclass:**
- `@dataclass(frozen=True)` for immutability
- Fields:
  - `name: str` - assembly name (required)
  - `bodies: tuple[Body, ...] = field(default_factory=tuple)` - all rigid bodies
  - `materials: tuple[Material, ...] = field(default_factory=tuple)` - material library
  - `ground_body: str = "ground"` - name of ground/world body

**Use tuples, not lists:** Frozen dataclass with list field allows list mutation. Tuples are truly immutable.

**Lookup methods:**
- `get_body(self, name: str) -> Optional[Body]`: Find body by name, return None if not found
- `get_material(self, name: str) -> Optional[Material]`: Find material by name, return None if not found

**validate() method (MODEL-06):**
Returns `list[str]` of validation errors. Empty list means valid.

Checks to perform:
1. Assembly name is required (not empty)
2. No duplicate body names
3. All body.material_name references exist in materials collection (if material_name is not None)
4. Bodies with inertia have positive mass (already enforced by Inertia, but verify)
5. Bodies with inertia have symmetric inertia tensor (np.allclose(I, I.T))

**Error message format:**
- "Assembly name is required"
- "Duplicate body names: {'name1', 'name2'}"
- "Body 'body_name' references unknown material 'material_name'"
- "Body 'body_name' has non-positive mass"
- "Body 'body_name' has non-symmetric inertia tensor"

**Imports:**
- numpy for tensor symmetry check
- typing for Optional, annotations
- Body, Material from same package
  </action>
  <verify>
```bash
python -c "
from inventor_exporter.model import Transform, Material, Inertia, Body
from inventor_exporter.model.assembly import AssemblyModel
import numpy as np

# Valid assembly
steel = Material(name='steel', density=7800)
body1 = Body(name='link1', transform=Transform(), material_name='steel')
body2 = Body(name='link2', transform=Transform(position=np.array([0, 0, 1])))
asm = AssemblyModel(name='robot', bodies=(body1, body2), materials=(steel,))

errors = asm.validate()
assert errors == [], f'Expected no errors, got {errors}'

# Lookup methods
assert asm.get_body('link1') == body1
assert asm.get_body('nonexistent') is None
assert asm.get_material('steel') == steel
assert asm.get_material('plastic') is None

# Missing assembly name
asm_no_name = AssemblyModel(name='', bodies=(body1,))
errors = asm_no_name.validate()
assert any('name' in e.lower() and 'required' in e.lower() for e in errors), f'Expected name error: {errors}'

# Duplicate body names
dup = AssemblyModel(name='test', bodies=(
    Body(name='same', transform=Transform()),
    Body(name='same', transform=Transform()),
))
errors = dup.validate()
assert any('duplicate' in e.lower() for e in errors), f'Expected duplicate error: {errors}'

# Unknown material reference
bad_ref = AssemblyModel(
    name='test',
    bodies=(Body(name='b1', transform=Transform(), material_name='unknown'),),
    materials=()
)
errors = bad_ref.validate()
assert any('unknown' in e.lower() and 'material' in e.lower() for e in errors), f'Expected material error: {errors}'

# Non-symmetric inertia tensor (bad data)
bad_inertia = np.array([[1, 0.5, 0], [0, 1, 0], [0, 0, 1]])  # Not symmetric
try:
    body_bad = Body(
        name='bad',
        transform=Transform(),
        inertia=Inertia(mass=1.0, inertia_tensor=bad_inertia)
    )
    asm_bad = AssemblyModel(name='test', bodies=(body_bad,))
    errors = asm_bad.validate()
    assert any('symmetric' in e.lower() for e in errors), f'Expected symmetry error: {errors}'
except Exception:
    pass  # Inertia might reject non-symmetric in __post_init__, which is also acceptable

print('AssemblyModel with validation: OK')
"
```
