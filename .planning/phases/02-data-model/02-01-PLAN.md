<task type="auto">
  <name>Task 2: Create Inertia dataclass with tensor transformations</name>
  <files>
    src/inventor_exporter/model/inertia.py
  </files>
  <action>
Create Inertia dataclass with parallel axis theorem and rotation support (MODEL-05).

**Inertia dataclass:**
- `@dataclass(frozen=True)` for immutability
- Fields:
  - `mass: float` (kg, required)
  - `center_of_mass: np.ndarray` (shape 3,, default zeros)
  - `inertia_tensor: np.ndarray` (shape 3x3, default zeros)
- Use `field(default_factory=...)` for numpy arrays
- `__post_init__` validates: mass >= 0, center_of_mass shape (3,), inertia_tensor shape (3,3)

**Methods (from research Pattern 4):**

`at_point(self, point: np.ndarray) -> np.ndarray`:
- Return inertia tensor transformed to a different point using parallel axis theorem
- Formula: `I' = I + m*[(d.d)E - d@d.T]` where d = point - center_of_mass
- This addresses MODEL-05: body origin may differ from center of mass

`rotated(self, R: np.ndarray) -> Inertia`:
- Return new Inertia with tensor rotated to new frame
- Tensor transforms as: `I' = R @ I @ R.T`
- CoM also transforms: `com' = R @ com`
- Return new Inertia instance (frozen, so can't modify self)

**Docstrings:** Document that inertia_tensor is expressed at center_of_mass position.
  </action>
  <verify>
```bash
python -c "
from inventor_exporter.model.inertia import Inertia
import numpy as np

# Create with mass only
i = Inertia(mass=1.0)
assert i.mass == 1.0
assert i.center_of_mass.shape == (3,)
assert i.inertia_tensor.shape == (3, 3)

# Test parallel axis theorem
# Sphere of mass 1 at origin, radius 1: I = 2/5 * m * r^2 = 0.4 on diagonal
I_sphere = np.diag([0.4, 0.4, 0.4])
inertia = Inertia(mass=1.0, center_of_mass=np.zeros(3), inertia_tensor=I_sphere)

# Move reference point by 1 meter along x
# Parallel axis: Ixx stays same, Iyy and Izz increase by m*d^2 = 1*1 = 1
I_at_1x = inertia.at_point(np.array([1.0, 0.0, 0.0]))
assert abs(I_at_1x[0, 0] - 0.4) < 1e-10, f'Ixx should stay 0.4, got {I_at_1x[0,0]}'
assert abs(I_at_1x[1, 1] - 1.4) < 1e-10, f'Iyy should be 1.4, got {I_at_1x[1,1]}'
assert abs(I_at_1x[2, 2] - 1.4) < 1e-10, f'Izz should be 1.4, got {I_at_1x[2,2]}'

# Test rotation - 90 deg about z should swap Ixx and Iyy
R_z90 = np.array([[0, -1, 0], [1, 0, 0], [0, 0, 1]], dtype=float)
I_ellipsoid = np.diag([1.0, 2.0, 3.0])
inertia2 = Inertia(mass=1.0, inertia_tensor=I_ellipsoid)
rotated = inertia2.rotated(R_z90)
assert abs(rotated.inertia_tensor[0, 0] - 2.0) < 1e-10, 'Ixx after rotation'
assert abs(rotated.inertia_tensor[1, 1] - 1.0) < 1e-10, 'Iyy after rotation'

# Invalid mass
try:
    Inertia(mass=-1)
    assert False, 'should raise for negative mass'
except ValueError as e:
    assert 'non-negative' in str(e).lower() or 'mass' in str(e).lower()

print('Inertia with tensor transformations: OK')
"
```
