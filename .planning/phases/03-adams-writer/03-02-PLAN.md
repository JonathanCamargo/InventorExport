  <action>
Create AdamsWriter that generates ADAMS View .cmd files:

1. Imports:
   - from pathlib import Path
   - from typing import Optional
   - import io
   - from inventor_exporter.model import AssemblyModel, Body, Material
   - from inventor_exporter.core.rotation import rotation_to_euler, EulerConvention
   - from inventor_exporter.writers.registry import WriterRegistry

2. Decorate class with @WriterRegistry.register("adams")

3. Class attributes:
   - format_name: str = "adams"
   - file_extension: str = ".cmd"

4. __init__(self, model_name: Optional[str] = None):
   - Store model_name override for ADAMS model naming
   - If None, will derive from assembly name in write()

5. write(self, model: AssemblyModel, output_path: Path) -> None:
   - Call model.validate() and raise ValueError if errors
   - Determine model name (override or from assembly name with model_ prefix)
   - Generate content sections in order: header, materials, rigid bodies, geometry
   - Write to output_path with utf-8 encoding

6. _get_model_name(self, model: AssemblyModel) -> str:
   - Use override if set
   - Otherwise: sanitize assembly name (replace space/dash with underscore)
   - Prefix with "model_" if not already prefixed

7. _generate_content(self, model: AssemblyModel, model_name: str) -> str:
   - Use io.StringIO for efficient string building
   - Write header comment with assembly name
   - Write "! === Materials ===" section
   - For each material: call _generate_material()
   - Write "! === Rigid Bodies ===" section
   - For each body: call _generate_rigid_body()
   - Write "! === Geometry ===" section
   - For each body with geometry_file: call _generate_geometry()
   - Return output.getvalue()

8. _generate_material(self, material: Material, model_name: str) -> str:
   - Convert density: kg/m^3 -> kg/mm^3 (multiply by 1e-9)
   - Default youngs_modulus to 207000.0 if None
   - Default poissons_ratio to 0.29 if None
   - Format as "material create" command with line continuations (&)
   - Use .15e format for density, .6e for youngs_modulus

9. _generate_rigid_body(self, body: Body, model_name: str) -> str:
   - Convert position: meters -> mm (multiply by 1000)
   - Get rotation as ZXZ Euler angles in degrees using rotation_to_euler()
   - Generate "part create rigid_body name_and_position" command
   - If body.inertia and body.material_name: call _generate_mass_properties()
   - Use .6f format for position/angles

10. _generate_mass_properties(self, body: Body, model_name: str) -> list[str]:
    - Convert inertia tensor: kg*m^2 -> kg*mm^2 (multiply by 1e6)
    - Generate "part create rigid_body mass_properties" command
    - Include: part_name, material_type, mass, ixx/iyy/izz/ixy/izx/iyz
    - Use .6f for mass, .6e for inertia components

11. _generate_geometry(self, body: Body, model_name: str) -> str:
    - If geometry_file is None, return empty string
    - Get STEP filename from body.geometry_file.name
    - Generate "file geometry read" command
    - Include: type_of_geometry=stp, file_name, part_name

Use patterns from 03-RESEARCH.md "Complete AdamsWriter Implementation".
