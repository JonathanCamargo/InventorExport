  <action>
Create mesh conversion utility following the pattern from RESEARCH.md.

Module structure:
1. `convert_step_to_stl(step_path: Path, stl_path: Path, tolerance: float = 0.1) -> None`
   - Single file conversion function
   - Uses cadquery.importers.importStep() and cadquery.exporters.export()
   - Export as binary STL (not ASCII - smaller files)
   - tolerance parameter controls mesh quality (lower = finer)

2. `MeshConverter` class for batch conversion:
   - `__init__(self, output_dir: Path, mesh_subdir: str = "meshes")`
   - `convert(self, step_path: Path, mesh_name: str) -> Path`
     - Returns relative path from output_dir to mesh file
     - Skips conversion if STL already exists (caching)
     - Creates meshes/ subdirectory if needed
   - `get_mesh_path(self, mesh_name: str) -> Path`
     - Returns relative path for mesh reference in XML

Handle cadquery import error gracefully:
```python
try:
    import cadquery as cq
    CADQUERY_AVAILABLE = True
except ImportError:
    CADQUERY_AVAILABLE = False
```

If cadquery not available, raise clear error when conversion attempted:
`raise RuntimeError("cadquery is required for mesh conversion. Install with: pip install cadquery")`

Include logging for conversion progress (debug level).
